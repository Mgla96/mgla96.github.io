Skills

The languages I am most proficient in are C++ and Python, followed by Javascript and Java. I have experience developing websites and web applications working with the full-stack as well as database and user interface design. The simulation game I built for the Chemistry department at UC Santa Barbara is an example of one of these web applications built with Node.js which you can view here. Another example is the Python Flask web application I built to solve the problem of fitness equipment being very scarce and hard to find due to COVID-19. This application was utilized by over 3.25k users from July-September and the source code can be viewed here.

I also have experience with Python in developing Machine Learning models, AI programs, automation scripts, and embedded software. An example of automation scripts I have created were for researchers working in the Ocean Recoveries Lab at UC Santa Barbara. They needed to measure the growth of coral reefs through converting their photos taken in the field to correctly scaled 3D models. This process required over 10 steps for every model and researchers had to be on standby to do the next step after model rendering had completed which could take numerous hours to render one model. I created two python scripts that automated this process for all models so only 1 manual step was required in total saving the researchers valuable time. The source code for this project can be viewed here.
